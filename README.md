# Quickstart AI Agent

This repository contains an AI agent built and deployed using **Databricks Mosaic AI Agent Framework**.

## About

This agent was created using the Databricks "Build your first AI agent" quickstart template. It demonstrates how to build, deploy, and serve an AI-powered agent on Databricks.

## Files

- `quickstart_agent.py` — Main agent code
- `Build your first AI agent` (notebook) — Step-by-step notebook used to build and deploy the agent

## Deployment

The agent is deployed as a Databricks Model Serving endpoint:
- Endpoint name: `agents_workspace-default-quickstart_agent`
- Can be tested via the Databricks Review App

## Tech Stack

- Databricks Mosaic AI Agent Framework
- MLflow
- Python
